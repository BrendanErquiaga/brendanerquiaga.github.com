Hidden

Web Game : Expected Release Q1 2012

Solo Project


Responsibilities

  • Programming the game
  • Designing the game
  • Making levels
  • Outsourcing Sound/Texture assets

Description

Hidden is a 1rst person puzzle platformer for web release. It is built in Unity 3D and was made by just me! I decided to make this project during a Game Jam, focusing on a single mechanic and creating a game from there is always a challenge.

The player controls an unnamed bi-ped who is placed in a strange world where geometry only exists near light.
