Description

When we created Real Dedicated we decided we were sick of writing code twice. And while we love the Unity3D editor; it does have some flaws.

Using Google Code we are managing the library and plan to build it up over the next year during our many projects. The goal is to create simple, flexible, re-useable code. By reducing the amount of time we spend writing generic code; we can focus on making our games fun.

The RD Unity Library will be made publicly available and moved to a Git style version control by the end of the year.
